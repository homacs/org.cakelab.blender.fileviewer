This is a so-called file debugger used to analyse the data of a Blender 3D model stored in a file with extension .blend.

Features

Available

Display of File Header
Header attributes which are displayed: File location, byte order, address width and version.
Listing of Struct Types
The DNA of a file is read, all struct types are listed and documentation can be displayed.
Listing of Blocks
All blocks of a file are listed, each block in a file can be displayed with all header attributes and available documentation.

In Progress

Separated Editors
Java .Blend external source code documentation displayed for selected structs.
Struct Documentation
Java .Blend external source code documentation displayed for selected structs.

Planned

Hex Viewer
A viewer, displaying the raw file content in hexadecimal notation.
